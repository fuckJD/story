    久他便邀请我去他家吃饭，我欣喜赴约，随后发展成男女朋友关系。两个月后的某天知道他在异地有女友，

    我十分后悔和自责。一方面我是个思想相对保守的人，对于第一次的丢失感到羞愧，另外一方面，觉得我
    自己也有责任，为何那么单纯听信他人。他第二天解释说，大家是成年人，他这样也不算强奸，我是自愿
    的。我当时也觉得自己是被诱奸了，可是好像我也没有反抗得很激烈，何况觉得报警也没用。发生了这件
    事后我有半年的时间一直陷于自责中，觉得自己那么不完美，怎么会那么笨呢？是不是自己本身就很低贱。
